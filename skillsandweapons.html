SKILLS & WEAPONS

All Saiyan, Ninja & Wizard moves, as well as weapons

When you have gotten a character card, the actions of the skills(except KKGs) which are on your card can be found here. Skills on this page can also be bought using XPs. The XPs of skills are stated at the end of the move.

TYPES OF SKILLS

There are 6 different type of skills the skill type is stated in a bracket after the name of the skill. The 6 types are as follows;

(nom): Normal Skill
(atk): Attacking Skill
(time): Time-Based Skill
(def): Defensive Skill
(neg): Neglecting Skill
(gain): Gain Skill

Note: A card slot can only hold one skill, but when the slot holds a mode, it can contain 3 modes.
If a turn is skipped you lose 2hp but gain 2cp.
You can only use a particular skill once, unless it is stated in the description of the skill that it can be used more than once.
A skill can be wasted, which means, if it has no effect or if the move is used in a wrong situation, the move is counted but will have no effect in the battle.
To use a skill against a player, just type the skill name.
A move can be sold for half the XPs it is worth.

 WEAPONS

Weapons are kept in your skill card, a slot which holds a weapon can hold up to 3 weapon skills. Wepons are mainly used to conserve cp while still maintaining an attacking mindset.
To use a weapon, type the weapon name followed by ''equip'' and then the weapon skill.
If you then want to use the same weapon, if it is still equipped, there is no need to type eqiup again.
Weapons can be bought by any race.
